# NAME

remove - Remove cargo by its name

# SYNOPSIS

**remove** \[**-y**\|**\--skip-confirm**\] \[**-f**\|**\--force**\]
\[**-h**\|**\--help**\] \[*KEYS*\]

# DESCRIPTION

Remove cargo by its name

# OPTIONS

**-y**, **\--skip-confirm**

:   

**-f**, **\--force**

:   

**-h**, **\--help**

:   Print help

\[*KEYS*\]

:   The keys or names of the objects to remove
