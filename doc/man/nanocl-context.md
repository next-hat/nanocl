# NAME

context - Manage contexts

# SYNOPSIS

**context** \[**-h**\|**\--help**\] \<*subcommands*\>

# DESCRIPTION

Manage contexts

# OPTIONS

**-h**, **\--help**

:   Print help

# SUBCOMMANDS

context-list(1)

:   List contexts

context-use(1)

:   Set current context

context-from(1)

:   Create a new context from a file

context-help(1)

:   Print this message or the help of the given subcommand(s)
