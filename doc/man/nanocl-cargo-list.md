# NAME

list - List existing cargo

# SYNOPSIS

**list** \[**-q**\|**\--quiet**\] \[**-l**\|**\--limit**\]
\[**-o**\|**\--offset**\] \[**\--filters**\] \[**-h**\|**\--help**\]

# DESCRIPTION

List existing cargo

# OPTIONS

**-q**, **\--quiet**

:   Only show keys

**-l**, **\--limit**=*LIMIT*

:   Limit the number of results default to 100

**-o**, **\--offset**=*OFFSET*

:   Offset the results to navigate through the results

**\--filters**=*FILTERS*

:   Filters

**-h**, **\--help**

:   Print help
