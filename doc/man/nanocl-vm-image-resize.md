# NAME

resize - Resize a VM image

# SYNOPSIS

**resize** \[**\--shrink**\] \[**-h**\|**\--help**\] \<*NAME*\>
\<*SIZE*\>

# DESCRIPTION

Resize a VM image

# OPTIONS

**\--shrink**

:   Shrink the image

**-h**, **\--help**

:   Print help

\<*NAME*\>

:   Name of the VM image

\<*SIZE*\>

:   New size of the VM image
