# NAME

remove - Remove a VM image

# SYNOPSIS

**remove** \[**-y**\|**\--skip-confirm**\] \[**-h**\|**\--help**\]
\[*KEYS*\]

# DESCRIPTION

Remove a VM image

# OPTIONS

**-y**, **\--skip-confirm**

:   

**-h**, **\--help**

:   Print help

\[*KEYS*\]

:   The keys or names of the objects to remove
