# Job remove

## SYNOPSIS

**remove** \[**-y**\|**--skip-confirm**\] \[**-h**\|**--help**\]
\[*KEYS*\]

## DESCRIPTION

Remove job by its name

## OPTIONS

**-y**, **--skip-confirm**  

**-h**, **--help**  
Print help

\[*KEYS*\]  
The keys or names of the objects to remove
