# State remove

## SYNOPSIS

**remove** \[**-s**\|**--state-location**\] \[**-y**\|**--yes**\]
\[**-h**\|**--help**\] \[*ARGS*\]

## DESCRIPTION

Remove elements from a Statefile

## OPTIONS

**-s**, **--state-location**=*STATE_LOCATION*  
Path or Url to the Statefile

**-y**, **--yes**  
Skip the confirmation prompt

**-h**, **--help**  
Print help

\[*ARGS*\]  
Additional arguments to pass to the file
