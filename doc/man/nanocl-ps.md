# NAME

ps - Show processes

# SYNOPSIS

**ps** \[**-q**\|**\--quiet**\] \[**-l**\|**\--limit**\]
\[**-o**\|**\--offset**\] \[**\--filters**\]
\[**-n**\|**\--namespace**\] \[**-k**\|**\--kind**\]
\[**-a**\|**\--all**\] \[**-h**\|**\--help**\]

# DESCRIPTION

Show processes

# OPTIONS

**-q**, **\--quiet**

:   Only show keys

**-l**, **\--limit**=*LIMIT*

:   Limit the number of results default to 100

**-o**, **\--offset**=*OFFSET*

:   Offset the results to navigate through the results

**\--filters**=*FILTERS*

:   Filters

**-n**, **\--namespace**=*NAMESPACE*

:   Show all processes for the given namespace

**-k**, **\--kind**=*KIND*

:   Show all processes for the given kind

**-a**, **\--all**

:   

**-h**, **\--help**

:   Print help
